## Версия Unity

Использовать Unity `6000.3.8f1`.

## Структура ассетов

1. Необходимо удалять неиспользуемые ресурсы из проекта.
2. Все проектные ассеты, созданные командой, должны храниться в директории `Assets/_Project`.
3. Внутри `Assets/_Project` ассеты должны распределяться по доменам.
4. Все объекты, находящиеся в папках проекта, должны быть сгруппированы в подпапки.
5. Внешние кодовые зависимости, SDK и плагины размещаются в директории `Assets/Plugins`.
6. Готовые сторонние визуальные ассеты размещаются в отдельной директории `Assets/ThirdPartyArt` и не смешиваются с рабочими ассетами проекта.

## Именование ассетов

1. Запрещено использовать пробелы в именах папок и любых файлов внутри проекта.
2. Unity-объекты, например материалы, префабы и скрипты, должны использовать `PascalCase`.
3. Сторонние объекты, например файлы текстур и аудиофайлы, должны использовать `snake_case`.

## Сцены

1. Объекты сцены должны быть структурированы по смысловым обобщениям.
