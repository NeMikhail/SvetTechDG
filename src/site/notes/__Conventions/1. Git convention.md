## Ветки

Название ветки формируется по шаблону:

```text
ТипРаботы/Номер_таски_название-задачи
```

Допустимые типы работы:

- `Feature`
- `Fix`
- `Architecture`
- `GameDesign`
- `Art`
- `Import`

Пример:

```text
Feature/10-PlayerCreation
```

## Коммиты

1. Название коммита должно содержать краткое описание изменения.
2. В названии коммита разрешается использовать только английский язык.
3. Номер задачи можно указывать как часть названия, но нельзя использовать его как единственное содержание коммита.

## Ограничения

1. Запрещается помещать в основной репозиторий крупные ассеты без согласования.
2. Запрещается помещать в репозиторий отладочный, закомментированный или нерабочий код.
